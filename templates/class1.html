Hello World, or rather, Hello Python

A quick introduction to Python syntax, variable assignment, and arithmetic in Python.

Python Syntax

1. Indentation

Python uses indentation to indicate a block of code.

if 5 > 2:
print("Five is greater than two!")

2. Variables

In Python, variables are created when you assign a value to it. Python has no command for declaring a variable.

x = 5
y = "Hello, World!"

3. Comments

In Python, comments are created by using #.

# This is a comment
print("Hello, World!")

4. Print Statement

The print() function prints the specified message to the screen.

print("Hello, World!")

Variable Assignment

1. Variable Assignment

In Python, variables need not be declared or defined in advance, as is the case in many other programming languages. To create a variable, you just assign it a value and then start using it. Assignment is done with a single equals sign (=).

n = 300
print(n)  # Output: 300

2. Variable Types

Variables in Python are not subject to type restriction. In Python, a variable may be assigned a value of one type and then later re-assigned a value of a different type.

var = 23.5
print(var)  # Output: 23.5
var = "Now I'm a string"
print(var)  # Output: Now I'm a string

3. Object References

When you make a variable assignment in Python, you are creating an object reference.

n = 300
# Here, n is a name attached to the object 300.

4. Chained Assignment

You can also perform chained assignment, which allows you to assign the same value to several variables simultaneously.

a = b = c = 300
print(a, b, c)  # Output: 300 300 300

Python Arithmetic

1. Addition (+)

Adds values on either side of the operator.

a = 10
b = 20
print(a + b)  # Output: 30

2. Subtraction (-)

Subtracts right hand operand from left hand operand.

a = 50
b = 30
print(a - b)  # Output: 20

3. Multiplication (*)

Multiplies values on either side of the operator.

a = 7
b = 6
print(a * b)  # Output: 42

4. Division (/)

Divides left hand operand by right hand operand.

a = 8
b = 2
print(a / b)  # Output: 4.0
