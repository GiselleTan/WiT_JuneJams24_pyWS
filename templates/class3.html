Booleans & Conditionals

Learn about Boolean Values and Conditional Statements in Python.

Boolean Values

1. Understanding Boolean Values

In Python, the two Boolean values are True and False. They must be capitalized.

print(True)  # prints: True
print(False)  # prints: False

2. Boolean Operations

Python includes several operations that return Boolean values, including comparisons (==, !=, <, <=, >, >=) and logical operations (and, or, not).

print(5 == 5)  # prints: True
print(5 > 10)  # prints: False
print(True and False)  # prints: False
print(True or False)  # prints: True
print(not True)  # prints: False

3. Using Booleans in Conditions

Boolean values are often used in conditional statements, like if statements.

if 5 > 2:
    print("Five is greater than two!")  # This will be printed
if 2 > 5:
    print("Two is greater than five!")  # This will not be printed

4. Converting to Booleans

You can convert other data types to Booleans using the bool() function. By default, bool() returns False for values that are empty or 0, and True otherwise.

print(bool(0))  # prints: False
print(bool(1))  # prints: True
print(bool(""))  # prints: False
print(bool("Hello"))  # prints: True

Conditional Statements

1. Understanding the if statement

The if statement is the most basic form of conditional branching. It allows for the execution of a statement or group of statements based on the value of an expression. Here’s a simple example:

if 1 > 0:
    print("One is greater than zero")

2. The else clause

The else clause can be used along with the if statement to specify a block of code to be executed if the condition is False:

if 1 < 0:
    print("One is less than zero")
else:
    print("One is not less than zero")

3. The elif clause

The elif (short for ‘else if’) clause allows you to check multiple expressions for True and execute a block of code as soon as one of the conditions evaluates to True:

x = 20
if x < 0:
    print("x is less than zero")
elif x == 0:
    print("x is zero")
else:
    print("x is greater than zero")

4. Nested conditions

You can have if/else constructs within another if/else construct. This is called nesting.

if x <= 0:
    if x = 0:
        print("x is equal to zero")
    else:
        print("x is less than zero")
else:
    print("x is greater than zero")

5. Conditional expressions (Ternary Operator)

Python’s ternary operator allows you to use if/else statements inline within your code:

x = 10
message = "x is less than 20" if x < 20 else "x is not less than 20"
print(message)
