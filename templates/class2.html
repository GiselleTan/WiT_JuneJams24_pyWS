Creating & Calling Functions

Learn how to create and call your own functions.

Creating Functions

1. Define the function

Start with the def keyword to tell Python you're defining a function.

def

2. Name the function

After def, write the name of your function. Let's call our function greet.

def greet

3. Add parameters in parentheses

If your function needs any inputs, you can put them in parentheses after the function name. In our case, we want to greet a person, so we'll need their name as an input. We'll call this input name.

def greet(name)

4. End the first line with a colon

This tells Python that the next lines will be the body of the function.

def greet(name):

5. Write the function body

This is where you put the code that runs when the function is called. In our case, we want to return a greeting message. Python uses indentation to define blocks of code, so make sure to indent the function body.

def greet(name):
    return f"Hello, {name}!"

6. Call the function

Now that the function is defined, you can call it by using its name followed by parentheses containing the arguments. Here's how you can call the greet function with the argument "Alice".

print(greet("Alice"))

Calling Functions

1. Identify the function

You need to know the name of the function you want to call. For example, let's say we have a function named greet that we defined earlier.

def greet(name):
    return f"Hello, {name}!"

2. Prepare the arguments

Look at the function definition to see what arguments it requires. Our greet function requires one argument: name.

def greet(name):
    return f"Hello, {name}!"

3. Call the function

To call the function, write the function's name followed by parentheses. Inside the parentheses, provide the arguments. For the greet function, you could call it with the argument "Alice" like this.

greet("Alice")

4. Use the function's output

If the function returns a value, you can use this value in your code. For example, you could print the result of the greet function:

print(greet("Alice"))

Passing Functions as Arguments

1. Define the first function

We start by defining a function say_hello that takes one argument name and returns a greeting string.

def say_hello(name):
    return f"Hello, {name}!"

2. Define the second function

Next, we define another function greet that takes two arguments: a function func and a name. Inside greet, it calls func(name) and returns the result.

def greet(func, name):
    return func(name)

3. Prepare the arguments for the function call

Now that we have our functions defined, we need to prepare the arguments for the function call. In this case, our greet function requires a function and a name as arguments. We already have our say_hello function and let's use "Alice" as the name.

def greet(func, name):
    return func(name)

4. Call the function with another function as an argument

Finally, we call our greet function with say_hello as the function argument and "Alice" as the name argument.

print(greet(say_hello, "Alice"))

5. Run the code

When you run this code, the greet function calls say_hello("Alice"), which returns "Hello, Alice!". This result is then printed out.

print(greet(say_hello, "Alice"))  # prints: Hello, Alice!
