Jumping into The Land of External Libraries

Learning how to use imports and work with External Python Libraries.

Python Imports

1. What is an import?

An import is a way to use code from other modules or packages in your own program. A module is a file that contains Python code, and a package is a collection of modules. You can use the import keyword to access the functions, classes, variables, and other objects defined in another module or package. For example, you can import the math module to use its mathematical functions:

import math
print(math.sqrt(25)) # 5.0

2. How to import specific objects or everything from a module or package?

You can also use the from keyword to import specific objects from a module or package, or use the * symbol to import everything. For example, you can import the sqrt function from the math module like this:

from math import sqrt
print(sqrt(25)) # 5.0

Or you can import everything from the math module like this:

from math import *
print(pi) # 3.141592653589793

However, it is not recommended to use the * symbol, as it can cause name conflicts and make your code less readable.

3. How to give an alias to a module or an object that you import?

You can also use the as keyword to give an alias to a module or an object that you import. This can make your code shorter and avoid name conflicts. For example, you can import the numpy module as np and use it like this:

import numpy as np
a = np.array([1, 2, 3])
print(a) # [1 2 3]

4. What are some external libraries and how to install and use them?

Python has many external libraries that are not part of the standard library, but can provide additional features and capabilities. These libraries are often developed and maintained by the Python community, and can be found on the Python Package Index (PyPI), which is a repository of software for Python.

Some of the most popular external libraries for Python are:

  • numpy: a library for scientific computing and working with arrays and matrices
  • pandas: a library for data analysis and manipulation
  • matplotlib: a library for data visualization and plotting
  • scipy: a library for scientific and technical computing

To use these external libraries, you need to install them first. There are different ways to install external libraries, but one of the most common and convenient ways is to use the pip tool. Pip is a package manager that can install and manage packages from PyPI. You can use the pip install command to install a package, followed by the name of the package. For example, to install numpy, you can run this command in your terminal:

pip install numpy
