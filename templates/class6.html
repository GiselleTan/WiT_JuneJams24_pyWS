Working with Strings & Dictionaries

Working with 2 fundamental data types in Python.

Working with Strings

1. What is a string?

A string is a sequence of characters, such as "hello" or 'Python'. You can use single or double quotes to create a string, but they must match. For example:

name = "Alice"
greeting = 'Hello'

2. What can a string contain?

A string can contain any character, including letters, numbers, symbols, spaces, and escape sequences. Escape sequences are special characters that start with a backslash \ and have a specific meaning. For example, \n means a new line, \t means a tab, and \" means a double quote. For example:

sentence = "She said, \"I love Python.\""
print(sentence)

This code will print:

    She said, "I love Python."

3. How to access the characters of a string?

You can access the characters of a string by using their index, which starts from 0. You can use square brackets [] to get the character at a specific position. For example:

word = "Python"
print(word[0]) # P
print(word[5]) # n

You can also use negative indexes to get the characters from the end of the string. For example:

word = "Python"
print(word[-1]) # n
print(word[-6]) # P

4. How to get a substring from a string?

You can use slicing to get a substring from a string. Slicing uses the syntax [start:end:step], where start is the starting index, end is the ending index (exclusive), and step is the increment. For example:

word = "Python"
print(word[1:4]) # yth
print(word[:3]) # Pyt
print(word[3:]) # hon
print(word[::2]) # Pto
print(word[::-1]) # nohtyP

5. How to concatenate and repeat strings?

You can use the + operator to concatenate two strings, or the * operator to repeat a string. For example:

first_name = "Alice"
last_name = "Smith"
full_name = first_name + " " + last_name
print(full_name) # Alice Smith
    
word = "ha"
laugh = word * 3
print(laugh) # haha

6. How to get the length of a string and check if a substring is in a string?

You can use the len() function to get the length of a string, or the in operator to check if a substring is in a string. For example:

name = "Alice"
print(len(name)) # 5
print("A" in name) # True
print("Z" in name) # False

7. How to convert other data types to strings and insert values into a string?

You can use the str() function to convert other data types to strings, or the format() method to insert values into a string. For example:

age = 25
message = "I am " + str(age) + " years old."
print(message) # I am 25 years old.

name = "Alice"
greeting = "Hello, {}!"
print(greeting.format(name)) # Hello, Alice!

8. How to manipulate strings with built-in methods?

You can use many built-in methods to manipulate strings, such as upper(), lower(), replace(), split(), join(), strip(), find(), count(), etc. For example:

name = "Alice"
print(name.upper()) # ALICE
print(name.lower()) # alice
print(name.replace("A", "E")) # Elise
print(name.split("l")) # ['A', 'ice']

words = ["Hello", "world"]
print(" ".join(words)) # Hello world

sentence = "  This is a sentence.  "
print(sentence.strip()) # This is a sentence.
print(sentence.find("is")) # 3
print(sentence.count("e")) # 3

Working with Dictionaries

1. What is a dictionary?

A dictionary in Python is a data structure that stores key-value pairs, where each key is associated with a value. You can use dictionaries to store and retrieve information by using the keys as indexes. For example, you can use a dictionary to store the names and ages of your friends:

friends = {"Alice": 25, "Bob": 26, "Charlie": 27}

2. How to create a dictionary?

To create a dictionary, you can use curly braces {} and separate the key-value pairs by commas. You can also use the dict() function and pass a list of tuples as an argument. For example, the above dictionary can also be created like this:

friends = dict([("Alice", 25), ("Bob", 26), ("Charlie", 27)])

3. How to access the value of a key?

To access the value of a key, you can use square brackets [] and specify the key name. For example, to get the age of Alice, you can do this:

print(friends["Alice"]) # 25

4. How to add, update, and delete key-value pairs?

To add a new key-value pair to a dictionary, you can assign a value to a new key. For example, to add a new friend named David with age 28, you can do this:

friends["David"] = 28

To update the value of an existing key, you can assign a new value to the key. For example, to change the age of Bob to 27, you can do this:

friends["Bob"] = 27

To delete a key-value pair from a dictionary, you can use the del keyword and specify the key name. For example, to remove Charlie from the dictionary, you can do this:

del friends["Charlie"]

5. How to iterate over a dictionary?

To iterate over a dictionary, you can use a for loop and the items() method, which returns a list of tuples containing the key-value pairs. For example, to print each key and value in the dictionary, you can do this:

for key, value in friends.items():
    print(key, value)

6. How to check if a key exists in a dictionary and get the number of key-value pairs?

To check if a key exists in a dictionary, you can use the in operator. For example, to check if Alice is in the dictionary, you can do this:

print("Alice" in friends) # True

To get the number of key-value pairs in a dictionary, you can use the len() function. For example, to get the size of the dictionary, you can do this:

print(len(friends)) # 3
