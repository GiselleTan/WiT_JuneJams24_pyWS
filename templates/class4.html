A guide on Lists in Python

Exploring List indexing, slicing and mutation. in Python.

Indexing in Lists

1. Understanding Iterables

In Python, lists are a type of iterable. Iterables are objects that you can iterate over, meaning you can traverse through all the different elements contained within the object.

    randomItems = [4, 6, 2, 56, 23, 623, 453]  # This is a list
    for item in randomItems:
        print(item)  # This will print each item in the list

2. What is Indexing?

Indexing in Python is a way to refer to the individual items within an iterable by their position. In other words, you can directly access your elements of choice within an iterable and do various operations depending on your needs.

3. Zero-Indexed

In Python, objects are “zero-indexed” meaning the position count starts at zero. So, if there are 5 elements present within a list, then the first element (i.e., the leftmost element) holds the “zeroth” position, followed by the elements in the first, second, third, and fourth positions.

    fruits = ["apple", "grape", "orange", "guava", "banana"]
    print(fruits[0])  # This will print "apple"
    print(fruits[1])  # This will print "grape"

4. Python Index Method

The index of a specific item within a list can be revealed when the index() method is called on the list with the item name passed as an argument.

    fruits = ["apple", "grape", "orange", "guava", "banana"]
    print("Index of Apple: ", fruits.index("apple"))  # This will print "Index of Apple: 0"
    print("Index of Banana: ", fruits.index("banana"))  # This will print "Index of Banana: 4"

5. Python Index Operator

The Python Index Operator is represented by opening and closing square brackets: []. You can use it to access elements at a specific index.

    fruits = ["apple", "grape", "orange", "guava", "banana"]
    print(fruits[0])  # This will print "apple"
    print(fruits[3])  # This will print "guava"

6. IndexError

Remember, trying to access an index outside the range of the list (for example, fruits[10] when the list only has 5 elements) will result in an IndexError.

Slicing Lists

1. Understanding List Indexing

In Python, the first element of the list has the index number 0, and the last element of the list has the index number N-1, where N is the total number of elements in the list.

Lst = [50, 70, 30, 20, 90, 10, 50]
print(Lst[0])  # Output: 50 (first element)
print(Lst[-1])  # Output: 50 (last element)

2. Basic List Slicing

The syntax for list slicing in Python is Lst[Initial:End:IndexJump]. This returns a portion of the list from index Initial to index End, at a step size IndexJump.

Lst = [50, 70, 30, 20, 90, 10, 50]
print(Lst[1:5])  # Output: [70, 30, 20, 90]

3. Slicing with Positive Indexes

You can slice a list using positive indexes.

Lst = [50, 70, 30, 20, 90, 10, 50]
print(Lst[::])  # Output: [50, 70, 30, 20, 90, 10, 50]

4. Slicing with Negative Indexes

Negative indexing starts from the end of the list.

Lst = [50, 70, 30, 20, 90, 10, 50]
print(Lst[-7::1])  # Output: [50, 70, 30, 20, 90, 10, 50]

5. Skipping Elements with Step Parameter

You can use the step parameter to skip elements while slicing.

List = [1, 2, 3, 4, 5, 6, 7, 8, 9]
print(List[::2])  # Output: [1, 3, 5, 7, 9]

6. Reversing a List

You can reverse a list by using a negative step parameter.

List = [1, 2, 3, 4, 5, 6, 7, 8, 9]
print(List[::-1])  # Output: [9, 8, 7, 6, 5, 4, 3, 2, 1]

Mutating Lists

1. Appending

You can use the append method to add an element to the end of a list. For example:

L = [1, 2, 3]
L.append(4) # L is now [1, 2, 3, 4]

2. Inserting

You can use the insert method to insert an element at a specific index in a list. For example:

L = [1, 2, 3]
L.insert(1, 10) # L is now [1, 10, 2, 3]

3. Removing

You can use the remove method to remove the first occurrence of an element from a list. For example:

L = [1, 2, 3, 2]
L.remove(2) # L is now [1, 3, 2]

4. Popping

You can use the pop method to remove and return the last element of a list. For example:

L = [1, 2, 3]
x = L.pop() # x is 3 and L is now [1, 2]

5. Sorting

You can use the sort method to sort the elements of a list in place. For example:

L = [5, 2, 4, 1, 3]
L.sort() # L is now [1, 2, 3, 4, 5]
